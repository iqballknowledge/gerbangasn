# 3. Competency Mining

-   Mengidentifikasi nilai sila.
-   Membedakan implementasi setiap sila.
-   Memilih perilaku yang paling sesuai.
-   Memahami hubungan nilai dengan tindakan.

# 4. Decision Pattern Mining

Jawaban benar umumnya dipilih karena:

-   Paling sesuai dengan nilai sila.
-   Paling mencerminkan semangat Pancasila.
-   Tidak diskriminatif.
-   Mengutamakan kepentingan bersama.
-   Menghormati hak orang lain.
-   Mengandung implementasi nyata.

Jarang menggunakan pertimbangan prosedur birokrasi.

------------------------------------------------------------------------

# 5. Distractor Analysis

## Karakter Distraktor

-   Menggunakan nilai sila lain.
-   Terlihat baik tetapi tidak sesuai sila yang ditanyakan.
-   Perilaku positif namun kurang lengkap.
-   Perilaku terlalu ekstrem.
-   Perilaku diskriminatif.
-   Perilaku individualistis.

Sebagian besar distraktor masih tampak logis.

------------------------------------------------------------------------

# 6. Difficulty Analysis

Kesulitan soal berasal dari:

-   Membedakan implementasi antar sila.
-   Opsi sama-sama baik.
-   Perbedaan pada nilai yang diwakili.
-   Tidak cukup mengetahui bunyi sila.

# 9. Common Misconceptions

-   Semua perilaku baik pasti sesuai sila yang ditanyakan.
-   Toleransi selalu identik dengan sila pertama.
-   Gotong royong selalu identik dengan sila ketiga.
-   Musyawarah tanpa keadilan sudah cukup.
-   Bersikap baik otomatis mewakili seluruh nilai Pancasila.
